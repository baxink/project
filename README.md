## 项目总览

项目创建后，按下面格式逐个追加：

### 项目：<船名>
- 项目入口：[打开项目](./<船名>/README.md)
- 目录：./<船名>/
- 起止日期：YYYY-MM-DD → YYYY-MM-DD
- 当前阶段：建造 / 系泊 / 试航 / 发证 / 交船
- 当前进度：`██████░░░░`（2-5）
- 下一节点：<日期或说明>
- 备注：<可空>

---

## 使用方式

1. 在根目录直接创建一个新项目文件夹。
2. 复制 `templates/project-template/` 到该项目文件夹作为骨架。
3. 按阶段整理资料。
4. 先更新这里的总览，再按需更新 [详细看板](./dashboard.md)。

## 约定

- 每条船的项目文件夹直接放在根目录，方便打开后立刻看到。
- 项目目录尽量只用短横线 `-`，避免空格和特殊符号。
- 同一阶段若有不同现场日期，可以拆成多个子文件夹。
- 看板以纯文本 Markdown 维护，不用表格，保证在 VS Code、预览和记事本里都直观。
